(5.)ส่วนที่เหลือจะเป็นการลองใช้ Method แบบสั้นๆผมจะเขียนจากที่ผมเข้าใจแต่ละ Method เลยนะครับ
.reduce = เป็นการรวมข้อมูลให้เหลือตัวเดียวโดยใช้ได้หลายวิธี

.push = เพิ่มเข้าไปตัวสุดท้ายใน array
.pop = ลบเข้าไปตัวสุดท้ายใน array
.unshift = เพิ่มเข้าไปตัวแรกใน array
.shift = ลบเข้าไปตัวแรกใน array
.spice = เป็นการเพิ่ม item เข้าไปใน array โดยเลือกจุดที่จะเริ่มได้และเลือกจะลบ item หลังจากจุดเริ่มได้ด้วย syntax = .spice(start,deleteCount,'Item')
.slice = เป็นการตัดส่วนหนึ่งของ array ออก syntax = slice(start, end)
.fill = เหมือนเป็นการเปลี่ยนค่าใน array ให้เป็น values ที่กำหนดโดยเลือกจุดเริ่มและจุดสุดท้ายได้ syntax = .fill(values,start,end)
